Mac への sbt のインストール
-------------------------

### サードパーティからのパッケージを使ってのインストール

> **注意:** サードパーティからのパッケージは最新版を使っているとは限らない。
> 何か問題があれば、パッケージメンテナに連絡してほしい。

#### [Macports](http://macports.org/)

```
\$ port install sbt
```

#### [Homebrew](http://mxcl.github.com/homebrew/)

```
\$ brew install sbt
```
